Janus
=====

[IARPA Janus Program API](http://www.libjanus.org)

# Build Instructions

    $ git clone https://github.com/biometrics/janus.git
    $ cd janus
    $ mkdir build
    $ cd build
    $ cmake ..
    $ make

# Run Test Scripts
    $ cd ../scripts
    $ ./run_verification.sh
    $ ./run_search.sh
